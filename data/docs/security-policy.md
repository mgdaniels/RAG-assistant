# Information Security Policy

*Owner: IT Team — last updated 15 January 2026*

The essential security rules for everyone. These are mandatory.

## Passwords and authentication

- Use the company password manager, **Vault**, for all work accounts.
- Passwords must be **at least 14 characters** and **unique to each service** — never reuse a password.
- **Multi-factor authentication (MFA) is required on every account**. Use an authenticator app rather than SMS wherever possible.

## Your laptop

- All laptops have **full-disk encryption** enabled by IT.
- Your screen must **auto-lock after 5 minutes**, and you should lock it manually whenever you step away.

## Reporting lost or stolen devices

If a work laptop or phone is lost or stolen, **report it within 1 hour** to **it-help@northwind.example** and **security@northwind.example** so access can be revoked.

## Phishing and suspicious messages

If you receive a suspicious email, **do not click any links or open attachments**. Forward it to **security@northwind.example** and flag it in Slack `#security`. When in doubt, ask — reporting a false alarm is always fine.

## Handling data

We classify information into four levels. Handle each according to its label:

| Classification | Examples | Handling |
| --- | --- | --- |
| Public | Marketing site, public docs | No restrictions |
| Internal | Most internal docs, org charts | Staff only |
| Confidential | Customer data, contracts | Need-to-know, encrypted at rest |
| Restricted | Credentials, security keys | Strictly limited, in Vault only |

Never put Confidential or Restricted data into external tools or personal accounts.
